Implement a specification using Agent OS implementation workflow.

This command coordinates the full implementation process:

**Phase 0: Infrastructure**
- database-engineer: Design and create database schema

**Phase 1: Backend**
- api-engineer: Build API endpoints and services
- backend-verifier: Verify backend quality

**Phase 2: Frontend**
- ui-designer: Build user interface components
- frontend-verifier: Verify frontend quality

**Phase 3: Testing**
- testing-engineer: Create and run test suites

**Phase 4: Final Verification**
- implementation-verifier: Certify implementation complete

The workflow executes task groups in order, with verification at each stage.
